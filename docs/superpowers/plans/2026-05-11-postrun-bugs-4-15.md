**Goal:** Fix twelve bugs surfaced in the 2026-05-10/11 runs: (4) test fixtures polluting prod DB; (5) duplicate decision rows for same (symbol, date); (7) per-process counter mislabeled as "rolling 24h"; (8) Key Factors space-stripping regression; (9) stop-guard log appears after R/R panel; (10) DuckDB cache check stalls Phase 1 on first ticker; (11) DefeatBeta company-name match breaks on `, Inc.`/`(The)` suffixes; (12) Trade Report regenerates at every startup; (13) no drain mode when 120m shutdown timer fires mid-cycle; (14) Phase 1 summary print silently fails for non-bulleted reports; (15) NLTK download path triggers `pathsec.ZipFile` security warning.

**Architecture:**

1. **Test-data cleanup (Bug 4):** one-shot DELETE script + insert-time guard in `app/database.py:add_decision_point` driven by a `STOCKDROP_ALLOW_TEST_SYMBOLS` env flag.
2. **Dedup (Bug 5):** add a `UNIQUE(symbol, date(timestamp))` index via `decision_date` generated column; widen the screener blacklist window from "previous trading day" to a configurable 7-calendar-day window; make `add_decision_point` collapse to an UPDATE if a same-day row exists.
3. **Agent-quota persistence (Bug 7):** persist `(timestamp, agent)` tuples to a new `agent_calls` SQLite table; `snapshot()` filters by `now - 24h`. Keep the in-memory ring for performance but seed it from disk on startup.
4. **Key Factors normalize (Bug 8):** write a regression test using the SHOP-style broken string; trace which transform produced it; fix the offending step (most likely an over-aggressive markdown / citation pass on `key_factors` list items). Add a property-check guard that flags any factor with `<2 word chars` per `4+ visible chars`.
5. **Stop-guard log order (Bug 9):** the recompute already runs first in code, but `logger.info(...)` writes to stderr while the panel writes to stdout, so terminals interleave them wrongly. Convert the `[PM stop-guard]` line in `research_service.py` from `logger.info` to `print()` and flush; mirror in tests.
6. **DuckDB cache warm at startup (Bug 10):** add `run_duckdb_cache_warm()` background task in `main.py` that constructs one throwaway `_DBTicker("SPY").earning_call_transcripts()` after `init_db()`. Phase 1 then hits a warm cache.
7. **DefeatBeta name-match (Bug 11):** extend `_transcript_matches_company` to strip leading/trailing `The`, `(The)`, parenthetical suffixes, and apply same normalization to the LLM-extracted transcript head. Add fixtures for "Gap, Inc. (The)" / "Loews Corporation" / "Berkshire Hathaway Inc.".
8. **Trade Report cache-aware startup (Bug 12):** make `run_trade_report_update` check the cache age before generating; if the CSV is `<55min` old, skip the run. Same logic on first iteration so startup is non-blocking.
9. **Drain mode on shutdown (Bug 13):** thread a `shutdown_predicate` callable into `check_large_cap_drops`; check it between tickers in the main loop. On a True read, log "[Drain] Shutdown requested — finishing current ticker, skipping queue" and return. Sleep waits in `run_periodic_check` already break on event.
10. **Phase 1 summary fallback (Bug 14):** replace the "(Detailed report generated, see full output)" branch with a deterministic first-N-sentences fallback, so every ticker gets a printable summary.
11. **NLTK path (Bug 15):** set `NLTK_DATA=<repo>/.nltk_data` before any defeatbeta import, and pre-download `punkt_tab` at module import or via setup script. Whitelists the path under pathsec.

def add_decision_point(symbol: str, price: float, drop_percent: float, recommendation: str, reasoning: str, status: str = "Ignored",

```python
# scripts/cleanup_test_decisions.py
"""Delete synthetic TEST*/T8_* rows from decision_points. Idempotent.

Usage:
    python scripts/cleanup_test_decisions.py          # dry-run preview
    python scripts/cleanup_test_decisions.py --apply  # actually delete
"""
import argparse
import os
import sqlite3
import sys


DB_NAME = os.getenv("DB_PATH", "subscribers.db")
PATTERNS = ("TEST%", "T8\\_%")  # SQLite LIKE; escape underscore


def _matched_rows(cursor) -> list:
    cursor.execute(
        "SELECT id, symbol, drop_percent, recommendation, timestamp "
        "FROM decision_points "
        "WHERE symbol LIKE 'TEST%' ESCAPE '\\' OR symbol LIKE 'T8\\_%' ESCAPE '\\' "
        "ORDER BY timestamp DESC"
    )
    return cursor.fetchall()


def main() -> int:
    parser = argparse.ArgumentParser()
    parser.add_argument("--apply", action="store_true",
                        help="Actually delete (default is dry-run preview).")
    args = parser.parse_args()

    if not os.path.exists(DB_NAME):
        print(f"[cleanup] DB not found at {DB_NAME}. Nothing to do.")
        return 0

    conn = sqlite3.connect(DB_NAME)
    cur = conn.cursor()
    rows = _matched_rows(cur)
    if not rows:
        print("[cleanup] No TEST*/T8_* rows in decision_points. Nothing to do.")
        conn.close()
        return 0

    print(f"[cleanup] Found {len(rows)} synthetic rows:")
    for r in rows:
        print(f"  id={r[0]} symbol={r[1]} drop={r[2]} rec={r[3]} ts={r[4]}")

    if not args.apply:
        print("[cleanup] DRY-RUN. Re-run with --apply to delete.")
        conn.close()
        return 0

    cur.execute(
        "DELETE FROM decision_points "
        "WHERE symbol LIKE 'TEST%' ESCAPE '\\' OR symbol LIKE 'T8\\_%' ESCAPE '\\'"
    )
    deleted = cur.rowcount
    conn.commit()
    conn.close()
    print(f"[cleanup] Deleted {deleted} rows.")
    return 0


if __name__ == "__main__":
    sys.exit(main())
```

## Task 3: Bug 5 — Calendar-day dedup window + collapsing same-day inserts

**Why:** FN appears twice on 2026-05-05 (both PENDING with different prices) and CNA appears across consecutive days as PENDING then AVOID. def get_analyzed_companies_within_days(day_window: int) -> List[str]:
    """Return company names analyzed in the last `day_window` calendar days.
    Calendar-day window. Catches back-to-back-day re-analyses that the
    single-day blacklist (get_analyzed_companies_since(prev_trading_day))
    misses on Tuesday after a Monday run.

    try:
        conn = sqlite3.connect(DB_NAME)
        cursor = conn.cursor()
        cursor.execute(
            "SELECT DISTINCT UPPER(company_name) FROM decision_points "
            "WHERE company_name IS NOT NULL "
            "AND timestamp >= datetime('now', ?) "
            "AND status != 'Pending'",
            (f"-{int(day_window)} days",),
        )
        rows = [row[0] for row in cursor.fetchall()]
        conn.close()
        return rows
    except Exception as e:
        print(f"Error fetching analyzed companies (window={day_window}d): {e}")

"""Persistent agent-call counter backed by SQLite. Thread-safe.

Two views:
  - `total_cycle`: count since last reset_cycle() (in-memory only).
  - `total_rolling_24h`: count over the last 24h, persisted across restarts.

The cycle view stays in-memory because cycle boundaries don't survive a
crash anyway. The 24h view goes to disk so the label "rolling 24h" is
accurate even right after a fresh start.
"""

from __future__ import annotations

import threading
from collections import Counter, deque
from datetime import datetime, timedelta, timezone
from typing import Deque, Dict, Tuple

from app.database import record_agent_call, load_agent_calls_within_24h


def _parse_ts(ts_str: str) -> datetime:
    # SQLite returns naive UTC strings. Treat as UTC.
    return datetime.fromisoformat(ts_str.replace(" ", "T")).replace(tzinfo=timezone.utc)


class AgentCallCounter:
    def __init__(self, window: timedelta = timedelta(hours=24)):
        self._lock = threading.Lock()
        self._cycle: Counter = Counter()
        self._rolling: Deque[Tuple[datetime, str]] = deque()
        self._window = window
        self._hydrate_from_db()

    def _hydrate_from_db(self) -> None:
        try:
            rows = load_agent_calls_within_24h()
        except Exception:
            rows = []
        for ts, agent in rows:
            try:
                self._rolling.append((_parse_ts(ts), agent))
            except Exception:
                continue

    def record(self, agent: str) -> None:
        now = datetime.now(timezone.utc)
        with self._lock:
            self._cycle[agent] += 1
            self._rolling.append((now, agent))
            self._evict_locked(now)
        # SQLite write is outside the lock — DB has its own.
        record_agent_call(agent)

    def reset_cycle(self) -> None:
        with self._lock:
            self._cycle.clear()

    def snapshot(self) -> Dict:
        now = datetime.now(timezone.utc)
        with self._lock:
            self._evict_locked(now)
            return {
                "total_cycle": sum(self._cycle.values()),
                "total_rolling_24h": len(self._rolling),
                "by_agent": dict(self._cycle),
            }

    def _evict_locked(self, now: datetime) -> None:
        cutoff = now - self._window
        while self._rolling and self._rolling[0][0] < cutoff:
            self._rolling.popleft()


# Module-level singleton. Import from here at call sites.

"""Detect and sanitize alphanumerically-glued strings in PM Key Factors.

The 'glued' bug: a Key Factor that should read like
  'Q1 2026 delivered massive $100.7B GMV and 15% FCF margins.'
arrives as
  'Q12026deliveredmassive100.7BGMVand15100.7BGMVand15%FCFmargins'

Root cause is upstream (LLM output or a normalizer pass — under investigation),
but downstream we need a deterministic check so the user-visible output is
never the garbled form. If we detect a glued string we replace it with a
visible flag rather than silently trying to re-space — re-spacing is
ambiguous and would mask the actual bug.
"""

from __future__ import annotations

import re
from typing import List

# A string is 'glued' if it has at least one token of >= 24 characters
# with NO whitespace, AND that token contains both digits and letters.
# Real Key Factors break at sentence boundaries well before 24 chars and
# even formal abbreviations like 'NYSE:SHOP' are <12 chars.
_GLUED_TOKEN_RE = re.compile(r"\S{24,}")


def is_glued_text(text: str) -> bool:
    if not text:
        return False
    for token in _GLUED_TOKEN_RE.findall(text):
        has_letter = any(c.isalpha() for c in token)
        has_digit = any(c.isdigit() for c in token)
        if has_letter and has_digit:
            return True
    return False


def sanitize_factors(factors: List[str]) -> List[str]:
    """Replace any glued factor with a [FLAG] entry. Preserves order."""
    out: List[str] = []
    for f in factors:
        if isinstance(f, str) and is_glued_text(f):
            preview = f[:80]
            out.append(
                f"[FLAG] KEY_FACTOR_MALFORMED: glued/space-stripped output "
                f"from PM. Raw: {preview}..."
            )
        else:
            out.append(f)
    return out
```

    @staticmethod
    def _transcript_matches_company(transcript_text: str, expected_company: str) -> bool:
        """Defensive match: does the transcript text reference the expected company?

        Normalizes both sides to handle DefeatBeta's known ticker-collision
        bugs ('L' -> Loblaw instead of Loews) without false-rejecting valid
        transcripts where the screener name includes corporate suffixes
        (e.g. 'Gap, Inc. (The)') that don't appear in the call body.

        Returns True if no expected_company was provided (backward compat).
        """
        if not expected_company:
            return True
        if not transcript_text:
            return False

        def _normalize(name: str) -> str:
            n = name.lower()
            # Drop parenthetical suffixes: "(The)", "(Class A)", "(Holdings)"
            import re as _re
            n = _re.sub(r"\s*\([^)]*\)\s*", " ", n)
            # Drop trailing comma+The
            n = _re.sub(r",\s*the\s*$", "", n)
            # Drop leading "The "
            if n.startswith("the "):
                n = n[4:]
            # Strip common corporate suffixes
            for suffix in (
                " corporation", " corp.", " corp", " incorporated",
                " inc.", " inc", " plc", " ltd", " limited",
                " companies", " company", " co.", " holdings", " group",
            ):
                if n.endswith(suffix):
                    n = n[: -len(suffix)]
            return n.strip(" ,.")

        expected_norm = _normalize(expected_company)
        if not expected_norm:
            return True

        head = transcript_text[:1500].lower()

        # Match either the full normalized name or its first significant token.
        first_token = expected_norm.split()[0] if expected_norm.split() else ""
        if expected_norm in head:
            return True
        if len(first_token) >= 3 and first_token in head:
            return True
        return False
```

```python
"""Phase 1 competitive-agent summary extraction.

Two-stage:
  1. If the report contains a 'Summary & Key Points' (case-insensitive)
     section, take the first N bullets.
  2. Otherwise fall back to the first N sentences from the report body,
     skipping markdown headers and empty lines.
"""

from __future__ import annotations

import re
from typing import List

_HEADER_RE = re.compile(r"summary\s*&\s*key\s*points", re.IGNORECASE)
_SENT_END_RE = re.compile(r"(?<=[.!?])\s+")


def _bullets(text: str, max_bullets: int) -> List[str]:
    out: List[str] = []
    for line in text.splitlines():
        s = line.strip()
        if not s:
            continue
        if s.startswith("-") or s.startswith("*") or (
            len(s) >= 2 and s[0].isdigit() and s[1] in ".)"
        ):
            out.append(s)
            if len(out) >= max_bullets:
                break
    return out


def _sentences(text: str, max_sentences: int) -> List[str]:
    body = "\n".join(
        line for line in text.splitlines()
        if line.strip() and not line.strip().startswith("#")
    )
    body = body.strip()
    if not body:
        return []
    sentences = _SENT_END_RE.split(body)
    return [s.strip() for s in sentences[:max_sentences] if s.strip()]


def extract_competitive_summary(report: str, max_bullets: int = 3) -> List[str]:
    if not report or not report.strip():
        return []

    m = _HEADER_RE.search(report)
    if m:
        tail = report[m.end():]
        bullets = _bullets(tail, max_bullets)
        if bullets:
            return bullets

    # Fallback to first N sentences
    return _sentences(report, max_bullets)
```
